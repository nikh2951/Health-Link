# HealthLink - Unified Healthcare Platform

HealthLink is a professional, modern healthcare platform designed to bridge the gap between patients and doctors. It provides a seamless experience for booking appointments, managing medical records, and handling emergency requests.

## Features

- **Doctor Portal**: Manage availability, view upcoming appointments, and provide prescriptions.
- **Patient Portal**: Search for doctors by area and specialization, book appointments instantly, and track medical history.
- **Emergency System**: Get immediate assistance and temporary relief prescriptions from available experts.
- **Unified Health Records**: Access vitals, medication history, and past reports in one place.
- **Advanced UI/UX**: Professional dashboard with glassmorphism and modern design aesthetics.

## Technology Stack

- **Frontend**: React, TypeScript, Vite, Vanilla CSS.
- **Backend**: Node.js, Express.
- **Database**: SQLite3.

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (version 16 or later)

### Installation

1. Clone the repository:
   ```bash
   git clone [repository-url]
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the application:
   ```bash
   npm run dev
   ```

The application will be available at `http://localhost:3000`.
